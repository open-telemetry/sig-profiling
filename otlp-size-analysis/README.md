# OTLP Size Analysis

This directory contains the `./otel-sizes.py` marimo notebook.

To edit the notebook, run:

```sh
marimo edit ./otel-sizes.py
```

The `cmd/otlp-analyze/` tool was used to create the CSV referenced in the notebook. The original OTLP data is internal and not available, but PRs to this cmd are welcome and new CSVs can be published after they land.
